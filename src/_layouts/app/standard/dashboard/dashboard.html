Notice Board

This is an area that you can use as a general "notice board" for all your BigRedSky users. The message can be specific:"Don't forget that all the group certificates need to be distributed by 14th of July" or broader "Remember that help for BigRedSky can be accessed by clicking on the red question mark icon in the top right hand corner of each page". You can also set an expiry date for this message to ensure the system is not cluttered with old messages.We trust this will be a useful tool in your use of BigRedSky.
